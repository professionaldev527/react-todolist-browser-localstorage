# React Todo List (with Local Storage)

A simple, fast, and responsive Todo List application built using React and Vite. This app helps you manage your daily tasks and automatically saves them to your browser's local storage, ensuring you never lose your data even if you refresh or close the tab.

## 🚀 Features

* **Task Management**: Easily add, edit, delete, and mark tasks as complete.
* **Local Storage Persistence**: Your todo items are saved directly in the browser's `localStorage`, providing a seamless and persistent user experience without needing a backend database.
* **Lightning Fast**: Bootstrapped with [Vite](https://vitejs.dev/) for an extremely fast development and build experience.
* **Modern Styling**: Styled using [Tailwind CSS](https://tailwindcss.com/) for a clean, responsive, and mobile-friendly interface.

## 🛠️ Technologies Used

* **Frontend Framework**: React.js
* **Build Tool**: Vite
* **Styling**: Tailwind CSS & PostCSS
* **Language**: JavaScript (ES6+)
